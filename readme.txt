スライドバーを動かして評価値をリアルタイムで記録する奴
タブレットでやるので色々めんどかったからhtmlを書いてheroku上にサーバを立ててデプロイした
https://guarded-journey-75979.herokuapp.com


node.js のFileSaver入れてるけど使えなかったから直で関数読み込んだような記憶



ローカルで実行するには、

npm init

npm start

を実行して、 http://localhost:3000 にアクセスする

使い方は、
・左上の入力欄：保存するとき名前つける用
・スライダ：スライダ　動かすと動く
・Start Stop Save Resetボタン：記録開始、停止、保存、データのリセット
　Saveを押したら"(左上の入力).csv"がダウンロードされる。
　csvファイルの中身：一行のデータ 1つ目の項目がstartボタンを押した時刻、2つ目以降が開始時から200msごとの保存データ（0～100）
とりあえず使ってみりゃわかる

ソースコードはpublic/index.htmlにあるよ
